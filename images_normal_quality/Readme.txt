Readme

train/train_shabby - The training data for the modeling process.
train/train_clean - Ground truth of training data.

validate/validate_shabby - The validation data for the modeling process.
validate/validate_cleaned - Ground truth of validation data.

test/test_shabby - Shabby version of testing set to evaluate model performance.
test/test_cleaned - Ground truth of testing data.

competition_images - The shabby versions of the solutions. Denoise these documents, run your solution through make-submission.py, and submit the CSV file.
